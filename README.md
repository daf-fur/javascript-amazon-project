# Amazon Clone (Frontend)
A fully responsive replica of Amazon built using HTML, CSS, and JavaScript. This project focuses on creating a dynamic and interactive shopping experience, mimicking core UI features of a modern e-commerce platform.

## Lessons Learned 
- JavaScript event handling
- DOM Manipulation
- UI Design
